# Avinash Inapakurthi - Cyberpunk Developer Portfolio

A high-fidelity, interactive, and responsive cyberpunk-themed developer portfolio website built using React, Vite, TypeScript, Tailwind CSS v4, and GSAP animations.

## 🚀 Live Demo
Visit the live portal here: [https://avinashinapakurthi.vercel.app/](https://avinashinapakurthi.vercel.app/)

## 🎨 Theme & Visual Design
- **Theme**: Cyberpunk dark void (`bg-[#05050a]`) with neon cyan (`#00f0ff`) and blue accents.
- **Background**: Interactive HTML5 Canvas particle system that connects nodes, responds to mouse hover, and adjusts to viewport changes.
- **Aesthetics**: Glassmorphism cards, glowing borders (`box-shadow`), neon scanlines, and custom grid borders.

## 🛠️ Tech Stack
- **Frontend Framework**: React 19 (TypeScript)
- **Bundler**: Vite
- **Styling**: Tailwind CSS v4 (Vanilla CSS variables)
- **Animations**: GSAP (GreenSock Animation Platform) & ScrollTrigger for transition triggers
- **Icons**: Lucide React & Inline SVGs

## 📁 Key Features & Sections
1. **Interactive Hero Section**: Neon branding, floating 3D-tilt profile picture card, and a typing animation loop.
2. **About Me & System Metrics**: High-fidelity details featuring a custom cubic-ease stats counter (CGPA, Projects, Certifications, Internships).
3. **Skills Grid**: Technical skill cards grouped by category (Programming, Web, Backend, Data Science, Tools, Design) with animated progress bars.
4. **Experience Timeline (System Logs)**: Alternating timeline tracking education and work logs with path line animations.
5. **Certifications (Security Clearances)**: Verified certifications (AWS AI/ML, Data Analytics, Excel, GenAI) linked directly to LinkedIn.
6. **Projects (Deployed Systems)**: Interactive carousel showing 5 major projects:
   - **BloodConnect**: Emergency blood donation network with SOS broadcasting.
   - **Hospital Availability System**: Search and details for regional hospitals.
   - **Code Review Assistant**: AI-powered automatic syntax/flaw reviewing tool.
   - **AI Data Analyst**: Intelligent data exploration dashboard using natural language.
   - **Client Portfolio**: Sleek layout showcasing client business profiles.
7. **Contact Command Center**: Functional contact transmission form with input validation and submission states.

## 💻 Local Development Setup

To run the project locally:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/avinashinapakurthi6-glitch/avinash-inapakurthi.git
   cd avinash-inapakurthi
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Start development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```
